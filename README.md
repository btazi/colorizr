# COLORIZR

## Overview
Colorizr allows you to quickly set colors on your text (in your console).

## Instructions
`"this is a string".blue`

To get the entire list of colors:
`String.sample_colors`

## Installation
`gem build colorizr.gemspec`
`gem install colorizr-0.0.3.gem`

In your rb file:
`require 'colorizr'`
